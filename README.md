# Dancing Pipes 🎹🕺

Welcome to Dancing Pipes, a multimodal interaction system designed for non-expert organ playing! This project, developed for academic purposes at our university, combines the beauty of pipe organ music with the elegance of gesture control. With Dancing Pipes, users can effortlessly control a pipe organ using intuitive gestures.

## Project Overview ✨

Dancing Pipes is composed of four main components:

1. **Camera and Gesture Recognition (Client) 👁️**: Captures and interprets user gestures through the camera.
2. **Server 🖥️**: Manages communication between components and processes incoming data.
3. **Frontend Service 🌐**: Provides a user-friendly interface for system interaction and visualization.
4. **Organ Client with MIDI Sequencer 🎵**: Connects to the pipe organ and sends MIDI signals based on recognized gestures.
