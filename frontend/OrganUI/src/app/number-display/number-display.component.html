Commands

Currently, no commands have been received from the camera, or information about the given commands has been deleted. As soon as you give the first command using the camera, they will appear here immediately.

Commands that appear here can be of two colors:

This is successfully executed command!
Green indicates that the command was successfully executed and will take effect from the next bar of the performance
This is not executed command!
Red indicates that the command was given but will not be executed. There are several reasons why this might happen:
  • The sequencer is not connected to the server.
  • You are trying to issue other commands without first issuing a start command.
  • You are attempting to issue a start command again without first issuing a stop command to end the previous start command.
  • You are issuing a command that will not affect the operation of the organ. For example, if the number of manuals is already set to the minimum, attempting to decrease the number of manuals further will have no effect.
